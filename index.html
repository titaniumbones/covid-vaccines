Ontario Vaccination

These charts use data from the Open Covid Data Working Group to display vacicnation progress in Ontario and compare per-capita values with other Canadian provinces (using population data from Statistics Canada). See the link for more details on data sources. I am neither an epidemiologist nor a government official so please don't use this dasboard for anything other than personal curiosity. The code is on Github and is pretty primitive! The "at current rates" projection just averages the number of vaccinations over the last few days -- this number will presumably become more reliable as we move out of the holidays.
